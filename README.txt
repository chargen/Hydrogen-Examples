Hydrogen Framework Examples (http://hydrogen.dotbunny.com/)
==================================================

This is some example content of how to use the Hydrogen Framework.
It requires that you have the Hydrogen Framework installed already.

It can be downloaded from the project on GitHub:

https://github.com/dotBunny/Hydrogen

Or you can find the Unity Asset Store package here:

https://www.assetstore.unity3d.com/#/content/10311
